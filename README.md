# Squadly Brief App

Веб-застосунок для збору брифів і перегляду заявок через адмін-панель.

Встанови залежності:

```bash
npm install
```

4. Запусти проєкт:

```bash
npm start
```
## Як працює
- Головна форма: `http://localhost:3002/`
- Адмінка: `http://localhost:3002/admin/login`
- Після відправлення бриф зберігається у `data/briefs.json`.
- Адміністратор може переглядати, редагувати і видаляти записи.
